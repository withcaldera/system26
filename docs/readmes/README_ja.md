# `System26`

System26は、Appleのオンデバイス`SystemLanguageModel`フレームワークを試せるように設計されたネイティブユーティリティです。Appleエコシステム全体でローカルの基盤モデルのリアルタイムパフォーマンス指標を表示します。

## 目的

このアプリは、Apple Silicon上でのオンデバイス推論の様子を気軽に確認するためのツールです。開発者や愛好家は、クラウド接続に頼らずにパフォーマンスを把握できます。

## サポートされているプラットフォーム

*   **macOS:** デスクトップワークフロー向けに最適化されています。
*   **iOS (iPhone):** 完全にレスポンシブなモバイルインターフェース。
*   **iPadOS:** 分割ビューと大画面レイアウトをサポートしています。

## 機能

*   **リアルタイム性能ビュー:**
    *   **スループット:** 1秒あたりのトークン数 (TPS) で生成速度を測定します。
    *   **レイテンシ:** 最初のトークンまでの時間 (TTFT) をミリ秒単位で追跡します。
    *   **メモリ:** 推論中の常駐メモリ使用量を監視します。
    *   **熱:** デバイスの熱状態（公称、普通、深刻、重大）を報告します。
*   **モードプリセット:**
    *   **一般目的:** 標準的なテキスト生成。
    *   **コンテンツタグ付け:** 特化した抽出および分類タスク。
    *   **ライティングツール:** 校正および編集のシミュレーション。
    *   **要約:** テキスト凝縮テスト。
    *   **ライブ翻訳:** リアルタイム言語翻訳パフォーマンス。
*   **カスタマイズ:** さまざまなモデルの動作をテストするために、システム指示とプロンプトを完全に編集可能です。
*   **ローカリゼーション:** 10言語（英語、中国語、スペイン語、フランス語、ポルトガル語、ドイツ語、イタリア語、日本語、韓国語、ベトナム語）に完全ローカライズされています。

## 前提条件

*   **オペレーティングシステム:** macOS 26 (Tahoe)、iOS 26、iPadOS 26、visionOS 26以降。
*   **ハードウェア:** Apple Neural Engineを搭載したデバイス（Apple Silicon MシリーズまたはAシリーズチップ）。
*   **開発:** ビルドにはXcode 26+が必要です。

## 実行方法

1.  Xcodeで `System26.xcodeproj` を開きます。
2.  ターゲットデバイス（Mac、iPhone、またはiPad）を選択します。
3.  ビルドして実行 (Cmd + R)。
