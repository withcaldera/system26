# `System26`

System26은 Apple의 온디바이스 `SystemLanguageModel` 프레임워크를 직접 살펴볼 수 있도록 설계된 네이티브 유틸리티입니다. Apple 생태계 전반에 걸쳐 로컬 파운데이션 모델의 실시간 성능 지표를 보여줍니다.

## 목적

이 애플리케이션은 Apple Silicon에서 로컬 추론이 어떻게 동작하는지 가볍게 확인할 수 있는 도구입니다. 개발자와 매니아는 클라우드 연결 없이도 온디바이스 AI의 성능을 파악할 수 있습니다.

## 지원 플랫폼

*   **macOS:** 데스크톱 워크플로에 최적화되었습니다.
*   **iOS (iPhone):** 완전히 반응하는 모바일 인터페이스.
*   **iPadOS:** 분할 보기 및 대형 화면 레이아웃을 지원합니다.

## 기능

*   **실시간 성능 보기:**
    *   **처리량:** 초당 토큰 수(TPS)로 생성 속도를 측정합니다.
    *   **지연 시간:** 밀리초 정밀도로 첫 번째 토큰까지의 시간(TTFT)을 추적합니다.
    *   **메모리:** 추론 중 상주 메모리 사용량을 모니터링합니다.
    *   **발열:** 기기 발열 상태(정상, 보통, 심각, 위험)를 보고합니다.
*   **모드 프리셋:**
    *   **일반 목적:** 표준 텍스트 생성.
    *   **콘텐츠 태깅:** 전문 추출 및 분류 작업.
    *   **쓰기 도구:** 교정 및 편집 시뮬레이션.
    *   **요약:** 텍스트 압축 테스트.
    *   **실시간 번역:** 실시간 언어 번역 성능.
*   **사용자 지정:** 다양한 모델 동작을 테스트하기 위해 시스템 지침 및 프롬프트를 완전히 편집할 수 있습니다.
*   **현지화:** 10개 언어(영어, 중국어, 스페인어, 프랑스어, 포르투갈어, 독일어, 이탈리아어, 일본어, 한국어, 베트남어)로 완벽하게 현지화되었습니다.

## 전제 조건

*   **운영 체제:** macOS 26 (Tahoe), iOS 26, iPadOS 26, visionOS 26 이상.
*   **하드웨어:** Apple Neural Engine이 탑재된 기기(Apple Silicon).
*   **개발:** 빌드하려면 Xcode 26+가 필요합니다.

## 실행 방법

1.  Xcode에서 `System26.xcodeproj`를 엽니다.
2.  대상 기기(Mac, iPhone 또는 iPad)를 선택합니다.
3.  빌드 및 실행 (Cmd + R).
