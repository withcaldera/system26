# `System26`

System26 是一款原生实用工具，用来体验 Apple 的设备端 `SystemLanguageModel` 框架。它展示 Apple 生态系统中本地基础模型的实时性能指标。

## 目的

该应用帮助开发者和爱好者了解 Apple Silicon 上的本地推理表现，而无需依赖云连接。

## 支持平台

*   **macOS：** 针对桌面工作流进行了优化。
*   **iOS (iPhone)：** 完全响应式的移动界面。
*   **iPadOS：** 支持分屏和大屏幕布局。

## 功能

*   **实时性能视图：**
    *   **吞吐量：** 测量生成速度，单位为每秒 Token 数 (TPS)。
    *   **延迟：** 以毫秒级精度跟踪首 Token 时间 (TTFT)。
    *   **内存：** 监控推理期间的常驻内存使用情况。
    *   **热状态：** 报告设备热状态（正常、尚可、严重、危急）。
*   **模式预设：**
    *   **通用：** 标准文本生成。
    *   **内容标记：** 专门的提取和分类任务。
    *   **写作工具：** 校对和编辑模拟。
    *   **摘要：** 文本压缩测试。
    *   **实时翻译：** 实时语言翻译性能。
*   **自定义：** 完全可编辑的系统指令和提示，用于测试各种模型行为。
*   **本地化：** 完全本地化为 10 种语言（英语、中文、西班牙语、法语、葡萄牙语、德语、意大利语、日语、韩语、越南语）。

## 先决条件

*   **操作系统：** macOS 26 (Tahoe)、iOS 26、iPadOS 26、visionOS 26 或更高版本。
*   **硬件：** 配备 Apple 神经网络引擎的设备（Apple Silicon M 系列或 A 系列芯片）。
*   **开发：** 构建需要 Xcode 26+。

## 如何运行

1.  在 Xcode 中打开 `System26.xcodeproj`。
2.  选择您的目标设备（Mac、iPhone 或 iPad）。
3.  构建并运行 (Cmd + R)。
